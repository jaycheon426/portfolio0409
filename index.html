안녕하세요. 저는 데이터로 문제를 푸는 분석가입니다.

비즈니스·프로덕트 관점의 데이터 수집 체계 구축부터 SQL 기반 유저 행동 분석까지 주도하며 성장을 만들어온 5년 차 데이터 분석가 천지혜입니다.

조직이 같은 기준으로 지표를 해석하고 빠르게 의사결정할 수 있도록 Single View와 Taxonomy를 정리하고, 대시보드로 공유·정착시키는 일을 해왔습니다.

또한 퍼널·코호트·세그먼트 분석을 통해 개선 지점을 구조적으로 찾고, 실행 가능한 액션과 측정 지표까지 연결해 왔습니다.

대형 커머스 및 글로벌 브랜드 고객사를 대상으로 데이터 파이프라인 설계와 분석 프로젝트를 수행하며, 여러 유관 부서와의 협업으로 데이터 활용도를 높이고 실질적인 성과 개선을 이끈 경험을 바탕으로 고객 행동 분석 체계를 고도화하고 성장을 견인하는 핵심 인재가 되겠습니다.

  • UTM 전략·GTM 기반 이벤트/프로퍼티 스키마 설계를 합니다.
  • SQL(CTE/Join/Window)로 데이터를 추출·가공해 분석합니다.
  • Tableau 대시보드로 지표를 시각화하고 커뮤니케이션합니다.
